Для корректной работы БД нужно установить модуль pyperclip. 
== pip install pyperclip

Это простая база данных, которая хранит пароли от сайта, а так же их логин.
Для того чтобы начать пользоваться ей, запустите файл OUTPUT.
После этого программа, предложит вам выбор, что конкретно вы хотите сделать.
Для того чтобы безопасно использовать данную БД. Советую установить 7-ZIP или WinRar. Так вы сможете сохранить целостность ваших данных,
с помощью создания запароленного архива.
